# Employee Salary Analysis

## Objective
Analyze employee salary data to understand compensation patterns and identify factors influencing salary distribution.


## Dataset
Employee Salary Dataset (Kaggle)


## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn


## Steps Performed
1. Data Loading and Inspection
   - Checked dataset structure using `.info()`, `.shape()`, `.describe()`

2. Data Cleaning
   - Verified no missing values
   - Ensured correct data types
   - Removed duplicates

3. Exploratory Data Analysis (EDA)
   - Salary distribution analysis
   - Experience vs Salary analysis
   - Age vs Salary analysis
   - Gender-based comparison

4. Statistical Analysis
   - Calculated mean, median, min, max salary
   - Correlation analysis between variables

5. Visualization
   - Histogram (salary distribution)
   - Line plots (experience & age trends)
   - Boxplot (gender comparison)
   - Heatmap (correlation matrix)


## Key Insights

- Salary distribution is highly right-skewed due to extreme high earners
- Average salary (~₹20L) is much higher than median (~₹2.5L)
- Experience has the strongest influence on salary (correlation ≈ 0.69)
- Age is also positively correlated with salary (≈ 0.67)
- Strong correlation between age and experience (≈ 0.97)
- Salary growth is non-linear with sharp increases at higher experience levels
- No significant gender-based salary disparity observed
