# Weather Data Analysis

## Objective
Analyze historical weather data to identify temperature trends and seasonal patterns.


## Dataset
Historical Hourly Weather Dataset (Kaggle)


## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn


## Steps Performed
1. Loaded temperature dataset
2. Converted datetime column into proper format
3. Set datetime as index for time-series analysis
4. Handled missing values using forward fill
5. Selected a specific city for analysis
6. Calculated:
   - Daily average temperature
   - Monthly average temperature
7. Performed seasonal pattern analysis using month grouping
8. Identified extreme temperature values
9. Visualized:
   - Daily trends (line chart)
   - Monthly trends (line chart)
   - Seasonal patterns (bar chart)
   - Heatmap for temperature variations


## Key Insights

- Temperature shows strong seasonal patterns with cyclical trends
- Maximum temperature recorded: 310.24 K (~37°C)
- Minimum temperature recorded: 250.77 K (~-22°C)
- Monthly averages provide clearer trends compared to daily fluctuations
- Temperature patterns repeat annually, confirming time-series behavior
- Dataset is suitable for forecasting and predictive modeling
