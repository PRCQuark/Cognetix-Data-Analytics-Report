# Food Delivery Data Analysis

## Objective
Analyze food delivery demand and operational performance.


## Dataset
Zomato Restaurant Dataset (Bangalore)


## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn


## Steps Performed
1. Loaded dataset and cleaned column names
2. Handled missing values and removed duplicates
3. Converted rating column into numeric format
4. Used votes as a proxy for order demand
5. Analyzed:
   - Total demand (votes)
   - Average rating and popularity
   - Online ordering availability
6. Performed area-wise demand analysis
7. Identified:
   - Top-performing restaurants
   - High-demand locations
8. Detected operational issues using rating vs demand


## 📊 Key Insights

- High-demand restaurants receive up to 16,000+ votes
- Strong correlation between ratings and demand
- Locations like Sarjapur Road and BTM show high demand
- Online ordering significantly impacts visibility and sales
- Customers prefer premium dining experiences
- Data contains duplicate entries requiring cleaning
- Operational inefficiencies observed in some restaurants


## Limitations

- No time-based data available
- Cannot analyze delivery delays or peak hours
