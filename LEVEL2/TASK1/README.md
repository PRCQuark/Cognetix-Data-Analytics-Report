#  Healthcare Patient Data Analysis

##  Objective
Analyze patient data to identify common diseases and demographic risk factors.

##  Dataset
Healthcare Dataset (Kaggle)


## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn


## Steps Performed
1. Loaded healthcare dataset
2. Validated data types and cleaned missing values
3. Removed duplicates and standardized categorical values
4. Created age groups for demographic analysis
5. Analyzed disease frequency using "Medical Condition"
6. Performed demographic analysis:
   - Age group vs disease count
   - Gender vs disease count
7. Identified high-risk groups using grouped analysis
8. Visualized results using bar charts and heatmaps


##  Key Insights

- Elderly population has the highest disease occurrence
- Elderly females (8111) and males (7985) represent top risk groups
- Young adults also show significant disease cases
- Gender distribution is nearly equal across all groups
- Disease risk increases with age
- Large dataset ensures reliable analysis
